# Archived Context Noise

This folder collects documents that are useful as historical record but should
not be part of the default development context for `DG_TWFD_v3`.

Do not read files in this folder unless the user asks for a specific archived
document or a current implementation question cannot be answered from:

- `docs/ACTIVE_CONTEXT.md`
- `docs/experiments/DG_TWFD_v3/HANDOFF_2026-04-20.md`
- current `src/`, `configs/`, and `scripts/`

## Contents

- `root/`: old root-level planning and Mac migration notes.
- `docs/`: historical DGTD v3 round audits, patch notes, and verification logs.
- `experiments/map_branch/`: previous branch documentation retained for traceability.
- `experiments/DG_TWFD_v3/`: superseded checklist, validation, workflow,
  reconstruction prompt, and legacy long-form architecture note.
